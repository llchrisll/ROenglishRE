#
[ ギルドとは ]
プレイヤーが組織できる特別なグループです。
一時的な協力関係としての意味合いが強い
パーティーに対して、ギルドはより持続的かつ高度な
連帯関係を結べるグループの単位となります。
ギルドに所属するには、自分でギルドを結成する
方法と、すでに結成されているギルドに加入する
方法があります。
#

[ ギルドでできること ]
1.ギルドチャットの利用
　ギルドメンバーだけにメッセージを送れる
　ギルドチャットが使えるようになります。

2.ギルドスキルの取得
　ギルドメンバーが経験値を上納することで
　ギルドのレベルを上げることができます。
　ギルドレベルが上昇すると、ギルドスキルの取得が
　可能となります。

3.攻城戦への参加
　攻城戦に参加できるようになります。
　※砦のエンペリウムにダメージを与えるためには、
　　ギルドスキル「正規ギルド承認」を取得している
　　必要があります。
#

[ ギルド加入方法 ]
すでに結成されているギルドに加入したい場合、
入りたいギルドの、加入権限を持つメンバーから
勧誘してもらう（加入要請してもらう）必要が
あります。

1.ギルド加入要請をしてもらう
　ギルドメンバーを募集している人を探して、
　加入要請を出してもらいましょう。

2.ギルド加入要請を承諾する
　ギルド加入要請が届いたら、承諾することで、
　ギルドの加入が完了します。
#

[ ギルド結成方法 ]
新しく自分のギルドを作りたい場合、
以下の手順でギルドを結成することができます。

1.ギルドの結成には「エンペリウム」というアイテム
　を所持している必要があります。
　「エンペリウム」は以下のモンスターなどを
　倒した時、ドロップすることがあります。

　　輝く草、オークゾンビ、レクイエム、ミネラル、
　　エンジェリング、ゴーストリング、黄金蟲、
　　バフォメット

2.「エンペリウム」を所持した状態で
　ギルド結成ボタンをクリックして
　ギルドを結成できます。
　※チャットウィンドウに
　　/guild "ギルド名"
　　と入力することでもギルド結成可能です。

3.Alt+Gキーでギルド情報ウィンドウを呼び出せます。

4.加入権限を持つギルドメンバーが、勧誘したい
　キャラクターを右クリックし、
　表示されたメニューの「○○さんにギルド加入
　要請」を選択することで、ほかのキャラクターを
　ギルドに招待できます。

ギルドに加入するとこのヘルプは見れなくなります。
詳しくは公式サイトをご覧ください。

それでは、楽しいギルドライフを！
